# HIS (Hospital Information System)
- 병원 정보 시스템 
- 실제로 병원에서 사용하는 시스템
- cdw (Clinical Data Warehouse) : 병원 정보 시스템에서 수집된 데이터를 저장하는 데이터 웨어하우스
- edw (Enterprise Data Warehouse) : 기업의 모든 데이터를 저장하는 데이터 웨어하우스

# ODS
- 야간시간을 이용해서 HIS를 복사한 시스템

# PT(PACT Table)
- 환자의 정보를 저장하는 테이블

# LT(Lookup Table)
- 코드를 저장하는 테이블
